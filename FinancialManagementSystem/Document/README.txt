
FinancialManagementSystemはコンソールベースの入出金管理用のJavaアプリケーションです。
javaアプリケーションはJREをコンピュータにインストールすればどのコンピュータでも利用することが可能です。

注意:
    このソフトウェアは依存関係として「sqlite-jdbc-3.45.1.0.jar」を要求します。libフォルダにコピーしてから起動してください
    このソフトウェアはデフォルトユーザーを作成しないため、まず最初にユーザーを作成してください
    また、本ソフトウェアは一部のみ日本語対応しています。 
    このソフトウェアはあくまで手動での入出金を管理します、直接銀行口座を操作したりするものではありません。

しかし、仕様上FinancialManagementSystem自体は下記の理由によりデータの共有はできません。
1.データベースファイルはコンピュータのユーザディレクトリに記録されます
2.Linuxのminimalインストールなどで日本語環境が利用できない場合
(SSH経由でなら可能な場合がありますが、未検証です)

1に関しては、ユーザーデータの保護を目的として、ユーザーデータはユーザディレクトリでのみ管理され
それ以外の場所での管理は不能ですし、方法はありません。

インストール方法:
java-1.8.0-openjdk-1.8.0.222-2.b10が必要です。
別途インストールしてください（上記ファイルのjre/binを環境変数のファイルパスに設定してください

このソフトウェア本体のインストールは不要です。
起動には同封のバッチファイル「run.bat」を実行(Linuxの場合はrun.sh)するか
またはコマンドプロンプトで下記のコマンドを実行してください。

java -jar FinancialManagementSystem.jar

アンインストール方法:
アンインストールするには、解凍したすべてのファイルを削除してください
